# Personal Site | Software Development Portfolio  
This repository hosts my personal portfolio website, designed with a focus on clean architecture. It serves as a central hub for my technical projects, including frontend architecture and network security tools.   

## 🚀 Live Demo  
You can view the live site here: **[willyhengwenloong.github.io/personal-site/index.html](https://willyhengwenloong.github.io/personal-site/index.html)** 

## 🛠️ Technical Stack  
* **HTML5 & CSS3:** Semantic structure utilising CSS variables for maintainable design systems
* **JavaScript (ES6+):** Lightweight, vanilla implementation  
* **Deployment:** Automated hosting via GitHub Pages  

## 📂 Featured Projects
* **Unified System Dashboard:** A solution for information fragmentation in student workflow, emphasising low-latency data rendering.
* **SDN DDoS Protection:** Technical implementation of network security using Mininet and Python to mitigate real-time attacks. 

## ⚙️ Local Development  
To run this project locally:  
1. Clone the repository:
```bash
   git clone https://github.com/willyhengwenloong/willyhengwenloong.github.io.git
```
2. Navigate to the directory
```bash
    cd willyhengwenloong.github.io
```
3. Open `index.html` in your preferred browser.
